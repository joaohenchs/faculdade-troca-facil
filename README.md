# TrocaFácil

Plataforma para troca e doação de itens entre usuários.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (auth, database, realtime)
- shadcn/ui

## Rodando o projeto

```bash
npm install
npm run dev
```

## Funcionalidades

- Cadastro de itens para troca ou doação
- Sistema de propostas entre usuários
- Chat em tempo real após aceite
- Confirmação dupla para finalizar troca
